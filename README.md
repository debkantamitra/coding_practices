# Coding Practices

A collection of JavaScript practice exercises and machine coding implementations.

## Structure

### `javascript-practice/`
JavaScript fundamentals and advanced concepts:
- Variable declarations (var, let, const)
- Array methods (map, filter, reduce)
- Closures
- Currying
- Objects
- `this` keyword
- Call, Apply, Bind
- Promises and polyfills
- Debounce & Throttling
- Event propagation
- LRU Cache

### `machine-coding-practice/`
UI components and features:
- Accordion
- Autocomplete
- Carousel (React + TypeScript)
- Configurable Color Boxes
- Grid with Random Colors
- Holy Grail Layout
- Infinite Scroll (React + TypeScript)
- Pagination (vanilla JS & React)
- React Window Example
- Tabs
- Todo List

## Usage

Most projects are vanilla JavaScript - open `index.html` in a browser. React projects require `npm install` or `yarn install` followed by `npm run dev` or `yarn dev`.
