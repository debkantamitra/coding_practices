Main Content

This is the main content area of the holy grail layout. The holy grail layout is a classic CSS layout pattern that consists of:

  • A header at the top
  • A footer at the bottom
  • Three columns in the middle: two sidebars and a main content area

The layout is called "holy grail" because it was historically difficult to achieve with CSS, especially making all columns equal height and ensuring the footer stays at the bottom.
